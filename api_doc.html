Property Description Type
Error Returns an error object. MYSQL_ERR
State Returns the state of the MYSQL_CONNECTION object. MYSQL_CONNECTION_STATE
Hostname Returns the host name used to establish the current connection. String
Username Returns the user name used to establish the current connection. String
Password Returns the password used to establish the current connection. String
DbName Returns the name of the database associated with the current connection. String
AutoUpdate Returns/sets the auto update option which is used to determine if changes/additions to the recordset are automatically saved when changing rows, requering the dataset, closing the recordset, etc. Boolean
QueryString Returns the most recently executed query. String
Method Desription Return Type Parameter Parameter Type Parameter Description
SetOption Used to set the options that will be sent to the server when atttempting to make a connection.  Can called multiple times to set more than one option. None eOption MYSQL_OPTION Detemines which connection option is being set.
sArg String Optional argument (default = "") used to specify any additional information needed to set the option (not required for all options).
ClearOptions Clears any connection options previously set by the SetOption method. None None    
OpenConnection Attempts to connect to a MySQL server. MYSQL_CONNECTION_STATE sHostName String The host name of the MySQL server to connect to.
sUserName String The user name to use when connecting to the MySQL server.
sPassword String The password to use when connecting to the MySQL server.
sDbName String The database to use.  If this parameter is a zero-length string (""), no initial database is selected.
lPortNum Long Optional argument (default  = 3306) to specify the port number of the MySQL server to connect to.
lFlags MYSQL_FLAG Optional argument (default  = 0) to specify one or more options that affect the operation of the server (multiple options can be combined using '+').
CloseConnection Closes a connection previously made by the OpenConnection method. None None    
SelectDb Changes the database for the current connection. Boolean sDbName String The database to use.
Execute Executes an SQL query.  For row returning queries, the data can be accessed from the MYSQL_RS object that  is returned.  MYSQL_RS sSQL String SQL query to execute.
lRecordsAffected Long Optional argument (default  = 0) that can be used to obtain the number of rows affected by a query (INSERT, UPDATE, DELETE) or the number of rows returned by a query (SELECT).
QueryInfo Returns information about the effect of the most recently executed query. String None    
Flush Runs the MySQL FLUSH statement on the database. None FlushType MYSQL_FLUSH The type of FLUSH statement to run on the database.
LastInsertID Return the AUTO_INCREMENT value generated by the most recently executed query. Long None
Ping Checks if connection is still active.  If not, reconnect using the same parameters used to initially make the connection. None None

Property Description Type
State Returns the state of the MYSQL_RS object. MYSQL_RECORDSET_STATE
AffectedRecords Returns the number of rows affected by a query (INSERT, UPDATE, DELETE) or the number of rows returned by a query (SELECT). Long
QueryString Returns the query used to generate the recordset. String
RecordCount Returns the number of records in a row returning query (SELECT). Long
FieldCount Returns the number of columns in a row returning query (SELECT). Long
Fields Returns a  field object. MYSQL_FIELD
BOF Returns whether or not the MYSQL_RS obejct is at the BOF marker. Boolean
EOF Returns whether or not the MYSQL_RS obejct is at the EOF marker. Boolean
AbsolutePosition Returns or sets the position of a recordset's current record from 1 to RecordCount, or returns one of the MYSQL_ABSOLUTE_POSITION values. Long
EditMode Returns the edit mode of the result set. MYSQL_EDIT_MODE
Table Returns the name(s) of the tables in this result set.  If there are more that one table, the names will be separated by commas. String
TableCount Returns the number of tables in this result set. Long
Method Description Return Type Parameter Parameter Type Parameter Description
MoveFirst Moves the current records position to the first record in the recordset. None None    
MoveLast Moves the current records position to the last record in the recordset. None None    
MovePrevious Moves the current records position to the previous record in the recordset. None None    
MoveNext Moves the current records position to the next record in the recordset. None None    
CloseRecordset Closes the MYSQL_RS object and releases any resources. None None    
FindFirst Finds the first record containing specified data in the given field.  Returns the record number if a match if found otherwise -1 is returned. Long vField Variant Name or number (0 to FieldCount-1) of the field being searched.
vData Variant Data to search for in the given field.
FindMethod MYSQL_FIND See Enum definition.
FindLast Finds the last record containing specified data in the given field.  Returns the record number if a match if found otherwise -1 is returned. Long vField Variant Name or number (0 to FieldCount-1) of the field being searched.
vData Variant Data to search for in the given field.
FindMethod MYSQL_FIND See Enum definition.
FindNext Finds the next record containing specified data in the given field.  Returns the record number if a match if found otherwise -1 is returned. Long vField Variant Name or number (0 to FieldCount-1) of the field being searched.
vData Variant Data to search for in the given field.
FindMethod MYSQL_FIND See Enum definition.
FindPrevious Finds the previous record containing specified data in the given field.  Returns the record number if a match if found otherwise -1 is returned. Long vField Variant Name or number (0 to FieldCount-1) of the field being searched.
vData Variant Data to search for in the given field.
FindMethod MYSQL_FIND See Enum definition.
RsToHtmlTable Creates an HTML table based on the data in the record set. String None
AddNew Enables add mode so any field value "lets" will cause a record to be inserted when the Update method is called.  Returns -1 if add mode cannot be enabled. Long None
Update Updates any changes to the current row when EditMode is MY_EDIT_IN_PROGRESS or adds a new record when EditMode is MY_EDIT_ADD.  Return value is -1 if an update is not allowed otherwise the return value is the number of records affected in the database. Long None
CancelUpdate Cancels any edits in progress. None None
Delete Delete the current row.  Returns -1 if a delete is not allowed otherwise the return value is the number of records affected in the database. Long None
Requery Requeries the record set. None None
